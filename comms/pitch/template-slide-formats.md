# Pitch Deck Template — Slide Format Catalog

Template: `pitch-deck-template.pptx` (61 slides, single DEFAULT layout, all formatting in shapes)

## Title Slides
- **S01** — Big title center-left, date + image right, company name top-left
- **S02** — Company name top-left, big title bottom-left, minimal (dark-style)

## Mission / Tagline
- **S03** — Tagline center-left, images right
- **S09** — Left: image + overlay text area. Right: big mission text (28pt) + label
- **S10** — Full-bleed image, tagline overlay bottom-center (15pt bold)

## Table of Contents (S04–S08)
- 4-column grid (Problem / Solution / Product / Evidence), numbered 01–04
- Variants highlight different sections with images

## Problem Slides
- **S11** — Section divider: "Problem" + "01"
- **S12** — Image right, single text block left
- **S13** — Image right (3.2" wide), subtitle (12pt) + big headline (28pt bold) left
- **S14–S16** — 3-column card layout: "What?" / "Why?" / "Why now?" with images (variants)
- **S17** — **Stats-driven**: 3 big numbers (28pt bold) with supporting text (12pt), image right
- **S18** — Quote/evidence: pull quote + attribution, "Problem" label

## Solution Slides
- **S19** — Section divider: "Solution" + "02"
- **S20** — Image left (3.2" wide), subtitle (12pt) + body text (15pt bold) right
- **S21** — Full-bleed image, text overlay bottom-center

## Market / TAM-SAM-SOM (S22–S26)
- **S22** — Market potential text + citation
- **S23–S26** — TAM/SAM/SOM circles with dollar values (variants for emphasis)

## Business Model / Pricing (S27–S31)
- **S27** — Text-based business model description
- **S28–S30** — 3-tier pricing cards with images (variant colors)
- **S31** — 3-column pricing table with shapes

## Competitive (S32–S34)
- **S32** — Competitive advantages text
- **S33** — Competitor logo grid (8 slots)
- **S34** — Feature comparison matrix (checkmarks)

## Team (S35)
- 9-person grid (3x3): headshot + name + position, image panel right

## Features / Architecture (S36)
- 4-quadrant feature grid (category label + feature list each), image right

## GTM / Timeline (S37–S40)
- **S37** — Horizontal 3-phase timeline with numbered milestones, labels + dates
- **S38–S40** — Variants highlighting phases 1, 2, 3

## Traction / KPIs (S41)
- 4 metric boxes (Marketing, Revenue, Growth, Users) + image right

## Testimonials (S42)
- Two pull quotes with attribution

## Product Section (S43–S49)
- **S43** — Section divider: "Product" + "03"
- **S44–S47** — 3-feature showcase with images (variants)
- **S48** — Product description + bullet list left, screenshot right
- **S49** — Screenshot left (3.2" wide), 5-row feature list right (name + description pairs)

## Evidence / Screenshots (S50–S59)
- **S50** — Section divider: "Evidence" + "04"
- **S51** — Full screenshot (layered frames)
- **S53–S55** — Screenshot center + feature callout (positioned differently per variant)
- **S56–S59** — Dual screenshot layout (side by side)

## Closing
- **S60** — Mission tagline + contact info
- **S61** — Big "Thank you" + company name + email
